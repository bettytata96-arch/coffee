# Coffee Website

## Project Description
This project is a multi-page responsive website built as a final project for the *Introduction to Web Development* course.  
The website presents a coffee-related theme with different pages showcasing content, images, and user interaction.

## Features
- Multi-page website (Home, Gallery, Review)
- Responsive layout for desktop and mobile
- Styled using external CSS
- Interactive behavior using JavaScript
- Clean separation of HTML, CSS, and JavaScript files

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run the Project
1. Download or clone the repository.
2. Open the `coffee.html` file in any modern web browser.
3. Navigate through the pages using the links provided.

## Project Structure
- `coffee.html` – Main page
- `gallery.html` – Image gallery page
- `review.html` – Review/content page
- `styleB.css` – External stylesheet
- `script2.js` – JavaScript for interactivity
